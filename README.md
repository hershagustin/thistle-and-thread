# Thistle and Thread

## Responsive Navigation

**Technologies:** HTML, CSS, JavaScript

---

### Project Overview
In this individual project, I created a **responsive navigation menu** for a website that works seamlessly on both desktop and mobile devices. I utilized CSS techniques like **flexbox** for layout, **media queries** for responsiveness, and a **hamburger menu** for mobile screens. The project also includes custom fonts, dropdown menus, and proper layering using **z-index**.

---

### Features 
- Built the responsive navigation menu using **flexbox**, **z-index**, and dropdown menus.  
- Added a **hamburger menu** for mobile devices.  
- Styled the design using **custom fonts** and consistent UI patterns.

---

### Skills Used
- **HTML & CSS:** Built the layout, dropdown menus, and managed element positioning.  
- **JavaScript:** Added interactivity for the hamburger menu on mobile devices.  
- **Responsive Web Design:** Used **media queries** to adjust layouts for different screen sizes.
